# NADSOFT Student CRUD Application

A full-stack Student Management CRUD Application built using React.js, Node.js, Express.js, and PostgreSQL with live deployment on Render.

---

# Tech Stack

## Frontend
- React.js
- Bootstrap 5
- Axios
- SweetAlert2

## Backend
- Node.js
- Express.js

## Database
- PostgreSQL

## Deployment
- Render (Frontend + Backend + Database)

---

# Features

✅ Add Student  
✅ Update Student  
✅ Delete Student  
✅ Pagination  
✅ Subject & Marks Management  
✅ PostgreSQL Database Integration  
✅ REST APIs  
✅ SweetAlert Notifications  
✅ Responsive UI  
✅ Live Deployment
